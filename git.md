# Basic commands

## start from system
```
git init
```

## start from github
```
git clone repo_url
```

https://github.com/tarsstar/tarsstar.git

### sync/pull code from repo
```
git checkout branch
git branch
git pull
```

### local changes track and push to repo
```
git status
git add filename
git add . (to add all the file)
git commit -m "show work description"
git push
```

### Advanced functions
```
git merge
```
